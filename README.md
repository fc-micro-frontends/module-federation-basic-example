# module-federation-basic-example

- [Guide](https://2woongjae.notion.site/React-React-081d572755254539af0e8c06bd9b131e)

### Using Node.js Version 20.9.0

```
➜ nvm use
```

### Install Dependencies

```
➜ pnpm i
```

### 개발 모드 실행

```
➜ pnpm -w dev
```

### 빌드 및 서버 실행

```
➜ pnpm -w build
➜ pnpm -w serve
```
